# Bloombox

A music player for you favorites files

## Compatibilities

- OS : Windows
- Files formats : MP3, MPEG, OPUS, OGG, OGA, WAV, AAC, CAF, M4A, MP4, WEBA, WEBM, DOLBY, FLAC
- Metadatas formats (read only) : ID3v1, ID3v2, MP4, FLAC

## Install dependencies


```npm install```

or

```yarn```
